# Blood Donation & Requests Hub

## Overview
This project is a system designed to manage blood donors and blood requests efficiently.

## Features
- Register and manage donors
- Handle blood requests using Queue
- Store and organize data using Linked List
- Search and track donor information

## Technologies
- Java
- Data Structures (Linked List, Queue)
